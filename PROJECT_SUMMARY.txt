🚀 QUICK START
═══════════════════════════════════════════════════════════════════

FOR WINDOWS USERS:
  1. Double-click: setup.bat
  2. Follow prompts to create admin account
  3. Run: python manage.py shell < create_categories.py
  4. Run: python manage.py runserver
  5. Visit: http://127.0.0.1:8000

FOR LINUX/MAC USERS:
  1. Run: chmod +x setup.sh && ./setup.sh
  2. Follow prompts to create admin account
  3. Run: python manage.py shell < create_categories.py
  4. Run: python manage.py runserver
  5. Visit: http://127.0.0.1:8000

MANUAL SETUP:
  1. python -m venv venv
  2. venv\Scripts\activate (Windows) or source venv/bin/activate (Linux/Mac)
  3. pip install -r requirements.txt
  4. python manage.py makemigrations
  5. python manage.py migrate
  6. python manage.py createsuperuser
  7. python manage.py shell < create_categories.py
  8. python manage.py runserver
