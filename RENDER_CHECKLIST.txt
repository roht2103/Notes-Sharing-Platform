🔧 COMMON FIXES
─────────────────────────────────────────────────────────────────
❌ Build Failed - "Permission denied: build.sh"
   ✅ Run: chmod +x build.sh && git add . && git commit -m "Fix" && git push

❌ "DisallowedHost" Error
   ✅ Check ALLOWED_HOSTS = ['.onrender.com'] in settings.py
   ✅ Push to GitHub, Render will redeploy

❌ Static Files Not Loading (No CSS)
   ✅ Check whitenoise in requirements.txt
   ✅ Check WhiteNoiseMiddleware in MIDDLEWARE
   ✅ Build logs should show "static files copied"
   ✅ Hard refresh: Ctrl + F5

❌ "ModuleNotFoundError: No module named 'gunicorn'"
   ✅ Add gunicorn==21.2.0 to requirements.txt
   ✅ Push to GitHub

❌ Database Connection Error
   ✅ Check DATABASE_URL environment variable
   ✅ Check psycopg2-binary in requirements.txt
   ✅ Verify database status is "Available"

❌ Site Takes 30-60 Seconds to Load
   ✅ This is normal for free tier (cold start)
   ✅ Use UptimeRobot to ping every 14 minutes (keeps warm)
   ✅ Or upgrade to paid plan ($7/month)

>>> print(get_random_secret_key())
>>> exit()

Copy output and paste as SECRET_KEY environment variable

🆘 TROUBLESHOOTING
─────────────────────────────────────────────────────────────────
If deployment fails:
1. Check Build Logs for errors
2. Verify all files pushed to GitHub
3. Check build.sh is executable (chmod +x)
4. Verify environment variables set correctly
5. Check requirements.txt has all packages
6. Test locally first with DEBUG=False
7. Check Render community forums

If site works but uploads don't persist:
- This is normal for free tier (ephemeral storage)
- Files deleted on restart/redeploy
- Solution: Add Cloudinary or upgrade plan
